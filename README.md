### How to copy and test the project using expo:

Clone the repository:
```
git clone https://github.com/felipesllopes/App-rickAndMorty.git
```

Navigate to folder:
```
cd App-rickAndMorty
```

Install the dependencies:
```
npm install
```

Run expo command:
```
npx expo start
```
